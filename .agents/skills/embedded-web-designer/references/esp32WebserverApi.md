# ESP32 WebServer API Reference

This reference covers the standard `WebServer.h` library (ESP32) and `ESP8266WebServer.h` (ESP8266) for serving compressed PROGMEM assets.

## Basic Setup
```cpp
#include <WebServer.h>
#include "index_html_gz.h"

WebServer server(80);

void setup() {
  server.on("/", HTTP_GET, []() {
    server.sendHeader("Content-Encoding", "gzip");
    server.send_P(200, "text/html", index_html_gz, index_html_gz_len);
  });
  server.begin();
}
```

## Key Methods

### `server.send_P(code, contentType, content, size)`
Serves content directly from PROGMEM.
- `code`: HTTP status (e.g., 200).
- `contentType`: MIME type (e.g., `"text/html"`).
- `content`: The PROGMEM array variable.
- `size`: The length variable generated by `convertToProgmem.py`.

### `server.sendHeader(name, value)`
Adds custom headers. **CRITICAL** for serving gzipped content:
```cpp
server.sendHeader("Content-Encoding", "gzip");
```

## MIME Types for Embedded Web
- HTML: `text/html`
- CSS: `text/css`
- JS: `application/javascript`
- JSON: `application/json`
- WebP: `image/webp`

## Handling REST/JSON
For Single-Page Applications, use `ArduinoJson` to parse incoming POST data:
```cpp
server.on("/save", HTTP_POST, []() {
  if (server.hasArg("plain")) {
    String body = server.arg("plain");
    // Parse JSON...
  }
  server.send(200, "application/json", "{\"status\":\"ok\"}");
});
```
